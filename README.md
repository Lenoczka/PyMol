# PyMol
# Визуализация и сравнение 3D-структур белков в PyMOL

## О проекте
Этот проект — практическое знакомство с инструментом молекулярной визуализации **PyMOL**.  
В рамках задания были проанализированы и визуально сравнены две 3D-структуры человеческого фермента **HEXA** (β-гексозаминидаза A), мутации в котором связаны с болезнью Тея-Сакса.

**Цель работы:**  
- Научиться управлять отображением белковых цепей  
- Изменять стили визуализации (cartoon, surface, licorice)  
- Выравнивать разные структуры друг на друга  
- Автоматизировать рутинные действия через **Python-скрипт** для PyMOL  

**Технологии:**  
`PyMOL`, `Python`

## Выполненные шаги (ручной режим в PyMOL GUI)

### 1. Загрузка и разделение цепей
- Скачаны PDB-файлы: `2GJX` (HEXA человека) и `2GK1` (альтернативная структура HEXA)
- Загружена структура `2GJX`
- Через `Display → Sequence` выделены цепочки **A** и **B**
- Через `Action (A) → copy to object` созданы отдельные объекты `chain_A` и `chain_B`
- Исходный объект `2GJX` скрыт (Hide everything)

**Результат:** `task9_img1.png` — видны только цепи A и B

### 2. Раскрашивание цепей
- Цепь A окрашена в **cyan**, цепь B — в **magenta** (инструмент `Color (C)`)

**Результат:** `task9_img2.png`

### 3. Разные стили отображения
- Цепь A отображена как **surface** (поверхность)
- Цепь B отображена как **licorice** (палочки)
- Отключён стандартный **cartoon** для обеих цепей (`Hide → cartoon`)

**Результат:** `task9_img3.png`

### 4. Выравнивание двух разных структур
- Загружена структура `2GK1`
- Из неё выделена цепь `I` в отдельный объект
- Выполнено **выравнивание** (align) цепи I из 2GK1 на цепь B из 2GJX
- Объектам заданы разные цвета
- Все остальные объекты скрыты

**Результат:** `task9_img4.png` — наглядно видно различие конформаций одного белка

## Автоматизация (бонусное задание №10)

Скрипт `task10_script.py` полностью воспроизводит шаги 1–4 **без участия пользователя** в GUI.

**Что делает скрипт:**
1. Загружает `2GJX.pdb`
2. Создаёт объекты `chain_A` и `chain_B`
3. Окрашивает их
4. Применяет разные стили (surface / licorice)
5. Загружает `2GK1.pdb`, выделяет цепь I
6. Выравнивает её на цепь B
7. Сохраняет все изображения (task9_img1.png … task9_img4.png)

**Как запустить:**
```bash
source ./.virtualenvs/PyCharmMiscProject/bin/activate
python task10_script.py
